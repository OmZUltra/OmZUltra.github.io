Image implementation HTML

An image can be diplayed with HTML using the <img/> tag. There are also some other purpose specific tags to assist the implementation of images. Here we will discuss the tags and attribut es required

HTML Tags
Name Tag Closing Tag Descriptions
Image <img/> No Used to render images on the HTML page.
Map <map> Yes Informs users that an interactive image is present with clickable areas.
Area <area/> No Enables the ability to specify the image area of the image map.
Tag Attributes
Name Attribute Reference Tag Descriptions
Source src="" <img/> Used to specify the url or local path of the image file to be displayed
Alternative Text alt="" <img/> Used to specify alternative text used to explain image content to both search engines and users.
Height height="" <img/> Used to specify the height of the image in pixels or percentages
Width width="" <img/> Specify width of the image in pixels or percentages.
Align align="" <img/> Where to align the image relative to other elements on the page.
Border border="" <img/> Specifies border thickness. (if any)
Vertical Space vspace="" <img/> Denotes spacing on the top and bottom of the image.
Horizontal Space hspace="" <img/> Denotes the spacing on the left and right side of the image
Name name="" <map> The name of the map between the image and the map itself.
